# Solver for Travelling Salesman Problem (TSP)

## Usage

### Input dataset format
The program takes a dataset formatted as follows:
```sh
NAME : sample_input
COMMENT: 5 POINTS
1 1 5 
2 3 2
3 4 3
4 5 7
5 6 11
EOF
```
### Output format
The program creates an output file formatted as follows:
```sh
tour cost: x
tour: [1,2,3,4,5]
```
